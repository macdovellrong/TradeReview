# 项目需求文档

## 1. 项目简介
本项目是一个K线回放复盘工具，目标是可以打开多窗口，进行多周期的K线同步回放

## 2. 核心功能
列出你希望实现的具体功能点：
- [ ] 绘图，多子窗口绘制不同周期K线图,这个窗口数量最好可以任意添加，不过一般不会超过4个
- [ ] 数据从本地加载，我已经有从QDM上下载了tick数据
- [ ] 导入CSV Tick数据并转存为高效数据库/文件格式（如Parquet/DuckDB）
- [ ] 自定义添加指标，比如EMA20 EMA30，布林带 MACD之类的
- [ ] 可以指定从任意时间开始继续回放，回放速度也需要可以控制
- [new] X轴需要显示时间，并且可以缩放
- [new] 图表可以选择周期
